Its marquee feature was the Grade Certification (段位認定) Mode, a challenge mode that plays 4 predetermined songs back-to-back with a shared life gauge — survive to the end to pass the grade. A randomized variant, Random Grade Certification, was also added. The game introduced a new Mode Select screen where players choose between Normal Mode, Grade Certification Mode, and Freedom Mode, the latter forgoing area progression in favor of uninterrupted continuous play. A Ticket system was added, with consumable items purchasable with credits that boost area progress and assist gameplay; one free ticket is awarded daily upon clearing a Grade Certification. The Friend Match mode was completely overhauled with a class rank system spanning 26 tiers (B5 through LEGEND), where defeating opponents earns class points and climbing to LEGEND grants special rewards.

The visual theme kept the bubbly "Splash" aesthetic while updating DX Pass character art. On the systems side, the DX Pass structure was revamped: Bronze and Silver passes were removed, Platinum was eliminated, and Gold pass pricing was reduced. Area progression boost was decoupled from passes and moved entirely to the new ticket system. The rating calculation was restructured, removing Grade Certification rating from the total and retaining only New Song and BEST ratings, with a new Silver-White (白銀) rating color tier added. A new partner character, 乙姫 (Otohime), became unlockable via a daily stamp card. Quality-of-life additions included a settings menu accessible from the mode select screen to skip optional gameplay stages, and a result screen color swap so that new records display in dark blue. The previously Friend Match–exclusive conduction song was also made available to all players.
