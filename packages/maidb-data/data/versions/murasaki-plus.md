maimai MURASAKi PLUS, released on June 22, 2017 as the follow-up to the MURASAKi version and represented by the kanji 菫, brought a major overhaul to the Event Course system alongside several new gameplay features. The event progression model was restructured from a simple play-count system to a stamp card format, where players earn stamps by participating in event courses — with bonus stamps awarded for specific achievements such as clearing a difficulty for the first time. Collectibles and rewards are now gated behind stamp card completion rather than raw play counts, and each course was extended to run for six weeks instead of the previous four. Additionally, songs from expired event courses became automatically available to all players without requiring prior play.

The update also introduced a new event difficulty tier called Ultra Expert, which randomly selects four Master difficulty charts from across the entire song library with no genre or level restrictions. Another major addition was Otohime's Room, a side activity where players spend MaiZENNY to give presents to the character Otohime, building affinity over time to unlock special rewards. System-wide changes included the return of Re:MASTER charts after a brief absence, the removal of the home ranker feature, a collection random-set function for profile customization, and improved CHUNITHM cross-game linkage with near-instant data sync.
